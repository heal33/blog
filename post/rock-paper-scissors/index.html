Rock, Paper, Scissors

2015/12/25

I just wanted a little more practice with JavaScript. I loved playing rock, paper, scissors (jan-ken-po) as a kid so I wanted to create that game in JavaScript and play against the computer..

As of right now, everything is done using prompts and alerts. Hopefully soon I’ll make it so the game can be played without them. But I feel as if I’m finally getting the hang of JavaScript. I was able to write this entire code by myself :)
