Bird Generator

2015/11/20

Working with javascript loops and functions. Check it out here!

View the code live on codepen.

Had to troubleshoot for 3 hours because the birds were not being reset each time the button was clicked.

After many hours of researching and trial and error, all I had to do was add html = “” at the end of the function to clear all birds images written to the html variable.

function myFunction() {
  var num = Math.floor(Math.random() * 1000) + 1;
  for (var i = 1; i <= num; i += 1) {
    html += "<img src='http://www.goldencob.com.au/media/2566/lovebird.png'>";
  }
  print(html);
  document.getElementById('birds-generated').innerHTML = num;
  //readys html to be cleared
  html = ""; 
}
